Assertion: chlorpromazine bioavailability continuous_value

Evidence
Evidence For (item 0) Evidence Type: EV_CT_Pharmacokinetic Pointer: 8157044 Reviewer: boycer
Quote:
Route of administration: oral

study duration: single dose

population: 11 male

ages:20-30

This study provides three values for f% (bioavailability) at different doses of chlorpromazine:

25mg : 8.07
50mg : 11.6
100mg : 18.4

The 25 and 50mg values are not statistically different from one another while the 100mg value is statistically different from the other two. The value entered in for this evidence item is the simple average of the 25 and 50mg dose rounded up to the nearest tenth.
Assumptions:
bioavailability: 0.1
